## 📦 Projects / Projetos
 
### ⛪ ChurchKit *(in progress / em desenvolvimento)*
 
A website platform designed for Baptist churches — giving communities a simple, modern, and affordable digital presence.
 
 
**Built for**
 
- Baptist churches and similar congregations / Igrejas batistas e congregações similares
- Communities that need a ready-to-use website without complexity / Comunidades que precisam de um site pronto, sem complexidade
**Stack:** Next.js · TypeScript · Prisma · TailwindCSS · Docker
